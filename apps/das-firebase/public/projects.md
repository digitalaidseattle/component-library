## AI Projects

The goal of this module is to allow rapid creation of repetitive content. Grant proposals, cover letters, or job applications can be generated with the aid of AI services. Cloning and/or altering the attributes of a project can be used to generate content tailored for different target audiences.

1. Name your project
2. Tell the AI model what you what generated
2. Add context files/text
3. Specify output
4. Generate the content

### Name your project

A unique name will aid in tracking and tracing additional changes

### Tell the AI model what you want generated

AI services use textual prompts to instruct its systems to generate content.  The project template will be used to form the prompt that will be sent to the AI service.

### Add context files/text

While optional, adding context files and/or text will help inform the AI service to adapt its responses to align with DAS's ibackground.  Text can include simple information like the url of DAS's website.  Files can include previous content examples.

### Specify output

If no output is  specified, the AI service will return content in the form of Markdown.  Otherwise the service will return structured JSON.  The structured response can be used for online forms which require sectioned answers to questions.

### Generate then content.

The DAS content generation system can use one of many LLM models to generate the content.
